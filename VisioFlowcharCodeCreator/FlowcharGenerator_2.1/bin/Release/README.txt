Примечания о работе
 1. Программа не обрабатывает:
 	- Иниц. списки : char ar[] = {...}; - Ошибка. Советую убрать сам список, например заменив на char ar[] = ...;  
	- Множественные команды на одной строке. Парсинг идет построчно, так что такие кострукции стоит заменить
		Например: if(true) printf("bla"); - Ошибка
			  if(true) - Все ок
			   printf("bla") - Все ок

УСТАНОВКА
1. После распаковки файлов, в папке с файлами запустите файл c расширением .VSTO для установки плагина на Visio
2. После успешной установки, зайдя в визио вы увидите работу плагина
3. Если захотите его выключить\включить в Visio, зайдите в Параметры->Надстройки->НадстройкиCOM->Перейти и установите галку
4. Если плагин при входе в Visio не запускается (даже если вы его не отключали), попробуйте включить его вручную по шагу 3
